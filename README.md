## Deep_Learning

This Repo contains different Deep Learning Projects that I conducted during a course at Hochschule Anhalt in Köthen (Anhalt).

Solar Panel Classification: Classification Task using Dense Networks, Batch Normalization, Dropout & ReLU activation.
CNN Classification: Image Classification Using CNN
RNN: Time Series Regression (weather data) with RNN/LSTM
NER: Named Entity Recognition on Newspaper Headlines

## Structure

data: Contains all training data except for images used in CNN due to their size
src: contains scripts for inference and modelling
models: models are saved here

## Additional Infos

The Projects were submitted as inference script + model in Codabench to use them on test data.
